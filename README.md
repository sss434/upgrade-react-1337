# Система управления заказами

React-приложение для управления заказами с использованием Mantine UI библиотеки.

## Возможности

- ✅ **Просмотр заказов** - таблица со всеми заказами, информацией о клиентах и статусах
- ✅ **Создание заказов** - форма для создания новых заказов с выбором товаров
- ✅ **Редактирование заказов** - изменение существующих заказов
- ✅ **Удаление заказов** - с подтверждением
- ✅ **Фильтрация** - по статусу, клиенту, поиск по тексту
- ✅ **Детальный просмотр** - модальное окно с полной информацией о заказе

## Технологии

- **React 19** - основной фреймворк
- **Mantine 7** - UI библиотека с готовыми компонентами
- **Vite** - сборщик и dev сервер
- **JavaScript** - язык программирования

## Установка и запуск

1. Установить зависимости:
```bash
npm install
```

2. Запустить dev сервер:
```bash
npm run dev
```

3. Открыть в браузере: http://localhost:5173

## Компоненты Mantine

Используются следующие компоненты из Mantine:

- `MantineProvider` - провайдер темы
- `Container`, `Paper` - контейнеры и карточки
- `Title`, `Text` - типографика
- `Button`, `Group` - кнопки и группировка
- `Table` - таблицы с прокруткой
- `Modal` - модальные окна
- `TextInput`, `Select`, `NumberInput` - формы
- `Badge` - бейджи для статусов
- `Grid`, `Stack` - лейауты
- `Space`, `Divider` - отступы и разделители

## Данные

Приложение использует моковые данные:
- **5 пользователей** с контактной информацией
- **10 товаров** разных категорий
- **10 заказов** с различными статусами

Все данные хранятся в памяти и сбрасываются при перезагрузке страницы.
